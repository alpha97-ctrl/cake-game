# cake-game

A simple Python game where the player picks three cake colors.  
The program checks if the chosen colors are available and responds accordingly.

## How to play

1. Make sure Python is installed.
2. Run the script:

```bash
python cake_game.py

Follow the prompts to pick three cake colors.
The program will tell you if your selection is valid.

Allowed Colors
BLUE, RED, GREEN, BROWN

Note: Input must match these exactly (uppercase).
